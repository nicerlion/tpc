== Description ==

'Display Product Variation' plugin allows you to show all the variations of a product on the main shop page. It also
override the default behaviour of Nav filters, correct variation product is shown matching to the filter selected.

== Plugin installation ==

Download and upload the plugin file to wordpress server. 
Go to: Admin console -> Plugins and activate 'WooComBros Display Product Variations' plugin.

== Plugin Configuration ==

To configure attributes to be shown, Go to: 
Admin Console -> Woocommerce -> Settings -> Products -> Display Variations and activate the attribute for display
variations.
